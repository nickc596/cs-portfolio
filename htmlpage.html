HTML stands for hyper text markup language. When we use HTML it used to write the webpage. in HTML we use things called tags. we write tags by first having a opening tag which starts with a less than sign and cloe it with a greater than sign just like this <>. after were done putting the signs in we put in a command a common tag is a p tag which is used for writing a paragraph just like this.

What is CSS

CSS stands for cascading style sheets. CSS is used to put life into the webpage like adding a picture or a gif and even color like a backgroup color and text colors. That is only the surface when you look at a webpage and all the links and the cool stuff that goes on it is probably CSS that makes all that happen.
